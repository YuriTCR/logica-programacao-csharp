# 🧮 Calculadora Simples em C#

Este projeto foi desenvolvido como parte dos meus estudos de lógica de programação utilizando C#.

## 📌 Objetivo

Praticar:

- Entrada de dados pelo console
- Estruturas condicionais (switch)
- Estrutura de decisão
- Tratamento de erro (divisão por zero)
- Organização básica de código

## ⚙️ Funcionalidades

A aplicação permite que o usuário:

1. Digite dois números
2. Escolha uma operação:
   - Soma
   - Subtração
   - Multiplicação
   - Divisão
3. Visualize o resultado da operação escolhida

## 🛠 Tecnologias utilizadas

- C#
- .NET (Aplicação Console)

## 📚 Conceitos aplicados

- Variáveis
- Conversão de tipos
- Estrutura `switch`
- Condições (`if`)
- Boas práticas básicas de organização
